# 📰 NewsBuddy

NewsBuddy is a responsive news aggregation web application that delivers the latest headlines from multiple categories using the GNews API. It provides users with a clean, interactive, and user-friendly interface to stay updated with current events from around the world.

Developed as part of my internship project, NewsBuddy demonstrates the use of HTML, CSS, JavaScript, and REST APIs to build a dynamic web application.

---

## 🚀 Features

### 📢 Real-Time News Updates
Fetches the latest headlines using the GNews API.

### 🗂️ Category-Based Browsing
Browse news across multiple categories:

- Top Stories
- World
- Business
- Technology
- Science
- Sports
- Entertainment
- Health

### 🔍 Search Functionality
Search articles by title or description.

### 🔄 Refresh News
Reload the latest articles instantly.

### 📱 Responsive Design
Optimized for desktops, tablets, and mobile devices.

### ⚡ Smart Caching
Reduces unnecessary API requests by storing previously loaded news.

### 🔗 Direct Article Access
Open full news articles from the original source with a single click.

## ⚙️ How It Works

1. User selects a news category.
2. JavaScript sends a request to the GNews API.
3. News articles are fetched and displayed dynamically.
4. Articles are cached for faster access.
5. Users can search through loaded articles.
6. Clicking "Read Full" opens the original news source.

## 🔑 API Configuration

This project uses the GNews API.

1. Create an account on GNews.
2. Generate your API key.
3. Open `script.js`.
4. Replace:

```javascript
const API_KEY = "YOUR_API_KEY";
```

with your own API key.
